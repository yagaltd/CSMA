Basic Accordion

Welcome to our component library. This accordion demonstrates collapsible content sections with smooth CSS-based animations.

Install via npm or include directly in your HTML. Full documentation is available in the docs folder.

Customize colors, spacing, and behavior using CSS variables and data attributes.

Keyboard Navigation

  • Arrow Up/Down: Move between items
  • Home: Go to first item
  • End: Go to last item
  • Enter/Space: Toggle item
