CSMA Datepicker

Two self-contained pickers that showcase the CSS-class reactivity pattern. The single picker supports keyboard navigation out of the box, while the range picker highlights the hovered span before you commit to an end date. Both instances emit DATE_SELECTED events through the EventBus and keep DOM updates contained to class changes.
