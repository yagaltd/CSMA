Chat Bubble Component

Message bubbles for conversations with avatars, headers, and color variants.

Type I: Pure CSS (No JavaScript Required)

Basic Chat Bubbles

Receiver Message (chat-start) - Left Side

Hi! How can I help you today?

Sender Message (chat-end) - Right Side

I have a question about CSMA architecture. 
<!-- Receiver message (Other person) - Left side -->
<div class="chat chat-start">
  <div class="chat-bubble chat-bubble-primary">Message</div>
</div>

<!-- Sender message (You) - Right side -->
<div class="chat chat-end">
  <div class="chat-bubble chat-bubble-secondary">Message</div>
</div>

Chat with Avatars

Hello! Welcome to CSMA Kit.

Nice to meet you! This component is awesome. 
<!-- Chat with avatar -->
<div class="chat chat-start">
  <div class="chat-image avatar">
    <img src="/avatar.jpg" alt="User">
  </div>
  <div class="chat-bubble chat-bubble-primary">
    Hello! Welcome to CSMA.
  </div>
</div>

CSS Classes Reference

  • .chat - Main container (grid layout)
  • .chat-start - Sender message (aligns left)
  • .chat-end - Receiver message (aligns right)
  • .chat-image - Avatar container
  • .chat-header - Name + timestamp header
  • .chat-footer - Status message (Delivered, Read)
  • .chat-bubble - Message bubble
  • .chat-bubble-primary|secondary|accent|info|success|warning|error - Color variants
